# Intranet Platform

Enterprise intranet platform for internal communication, document management, and ticket handling.

---

## 🚀 Vision
Build a modern, secure, and scalable intranet solution for organizations to:

- Centralize internal communication
- Manage departments and employees
- Handle internal support tickets
- Store and manage documents
- Provide role-based access control

Designed to be modular, user-friendly, and enterprise-ready.

---

## 🎯 Target Audience
- Small & Medium Businesses (SMB)
- Enterprise companies
- Educational institutions
- Corporate teams

---

## 🏗 Tech Stack

### Backend
- Python
- Django
- Wagtail CMS
- PostgreSQL

### Frontend
- Custom React frontend
- REST API integration

### Infrastructure
- Docker + Docker Compose
- Nginx (reverse proxy)
- CI/CD (GitHub Actions planned)

---

## 📦 MVP Scope
The first release (MVP) will include:

### Authentication & Roles
- User registration/login
- Role-based access (Admin, Manager, Employee)

### Departments
- Create/edit departments
- Assign employees
- Department page

### Employee Directory
- Employee profiles
- Search & filtering

### Documents Module
- Upload documents
- Access control
- Department-based visibility

### Ticket System
- Create internal tickets
- Status workflow (Open / In Progress / Closed)
- Assignment to employees
- Basic dashboard view
